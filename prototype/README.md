# 宠小满 Duxiaoman Pet AI 🐾

> **拍一拍，更懂它** — AI 宠物健康管家小程序原型

宠小满是一款面向城市养宠人群的 AI 宠物健康管理产品，通过多模态 AI（计算机视觉 + 大语言模型）为宠物提供 7×24 小时健康分析和养宠顾问服务。

## 🤖 AI Prompt 工作流

报告中包含 3 套生产级 Prompt 设计：

1. **宠物皮肤健康视觉分析** — 多模态图像 → 四级风险报告
2. **AI 养宠问答（症状分诊）** — 追问框架 → 分级判断 → 可操作建议
3. **宠物饮食建议** — 品种/年龄/健康 → 个性化喂养方案

详见 `宠小满_机会验证报告.md` 第 3.5 节。
